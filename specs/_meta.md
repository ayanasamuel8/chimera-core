# Technical Specification

## Data Models (Source of Truth)

### TrendSignal
The input data for the system.
```json
{
  "source": "string (e.g., 'twitter', 'google_trends')",
  "query": "string",
  "momentum_score": "float (0.0 - 1.0)",
  "timestamp": "iso8601 string"
}
```
## ContentDraft
The output generated by the agent.
```json
{
  "title": "string",
  "script_body": "string",
  "media_prompts": ["string"],
  "estimated_virality": "float"
}
```
